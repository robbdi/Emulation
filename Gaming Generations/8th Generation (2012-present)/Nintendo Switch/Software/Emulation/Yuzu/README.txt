EzYuzu
Yuzu Portable Updater for Standlone versions of Yuzu
https://github.com/amakvana/EzYuzu
https://yuzu-emu.org/



Menu Options
-----------------------------
New Install	Installs Yuzu & Redistributables. Resets configs & sets optimised GPU defaults
Upgrade		Upgrades Yuzu to the latest version, including Redistributables but excluding configs.
Yuzu		Updates Yuzu to the latest version, excluding Redistributables and configs.
Redist		Installs Redistributables. Helpful if Yuzu won't start after update.



Usage
-----------------------------
• Select your Yuzu root folder, click Update.
• It will fetch the latest Yuzu available, download it and extract to your Yuzu root folder.
• The Check button will check for an update before downloading.
• It should not overwrite your configs or games, but keep a backup before updating.



Yuzu Update
-----------------------------
1. Select your Yuzu root folder
2. Select Yuzu from the Download menu
3. Click the Check button to get update status (optional)
4. Click Update

It will download and extract the latest Yuzu portable.
e.g. https://github.com/yuzu-emu/yuzu-mainline/releases/download/mainline-0-322/yuzu-windows-msvc-20200721-86423219f.zip



Requirements
-----------------------------
• Latest version of .NET Framework installed.
• Administrator rights, to allow redistributables to install .
